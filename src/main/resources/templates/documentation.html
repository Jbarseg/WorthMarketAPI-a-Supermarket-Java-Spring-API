Products

Product Schema

All the information about products

Key Type Description
productId int It is the identifier of the product. It is specific to each of the products.
productName String Is the name of the product.
productCategoryId int It is the number of the category of which the product is part.
productPrice Double Is the price of the product.
productStock int It is the available quantity of the product.
productStatus Boolean It is to know if the product is available or not.

GET all products

To have access to all available products you have to use the /products/all endpoint

GET https://worthmarketapi-production.up.railway.app/api/products/all 
[
    {
        "productId": 2,
        "productName": "Mango",
        "productCategoryId": 1,
        "productPrice": 2100.0,
        "productStock": 250,
        "productStatus": true,
        "productCategory": {
            "categoryId": 0,
            "category": "",
            "categoryStatus": false
        }
    }
]

GET a specific product

To have access to a specific product you have to use the /products/{id} endpoint

GET https://worthmarketapi-production.up.railway.app/api/products/{id} 
[
    {
        "productId": 48,
        "productName": "Cereal",
        "productCategoryId": 8,
        "productPrice": 7000.0,
        "productStock": 75,
        "productStatus": true,
        "productCategory": {
            "categoryId": 0,
            "category": "",
            "categoryStatus": false
        }
    }
]

SAVE a new product

To post a specific product you have to use the /products/save endpoint. You have to add a body in some format with Postman or other API platform.

POST https://worthmarketapi-production.up.railway.app/api/products/save

You have to add a body like this:

[
    {
        "productName": "Cereal",
        "productCategoryId": 8,
        "productPrice": 7000.0,
        "productStock": 75,
        "productStatus": true,
    }
]
Status: 200 OK

DELETE a product

To delete a specific product you have to use the /products/delete/{id} endpoint

DELETE https://worthmarketapi-production.up.railway.app/api/products/delete/{id}
true

Purchases

Purchase Schema

All the information about purchases

Key Type Description
purchaseId int It is the identifier of the purchase. It is specific to each of the purchases.
clientId int It is the identifier of the client. It is specific to each of the clients.
purchaseDateTime int It is the date the purchase was made.
purchasePaymentMethod Double It is the payment method used for the purchase.
purchaseComment int It is a comment on the purchase.
purchaseStatus Boolean It is to know if the purchase was made or not.
purchaseItems none It is the introduction for all the information about the purchase items.
totalCost Double It is the total cost for all items in the purchase.
itemStatus Boolean It is to know if the items are available or not.

GET all purchases

To have access to all available purchases you have to use the /purchases/all endpoint

GET https://worthmarketapi-production.up.railway.app/api/purchases/all 
[
    {
        "purchaseId": 1,
        "clientId": "4546221",
        "purchaseDateTime": "1992-08-10T17:30:00",
        "purchasePaymentMethod": "E",
        "purchaseComment": "",
        "purchaseStatus": "P",
        "purchaseItems": [
            {
                "productId": 1,
                "stock": 10,
                "totalCost": 3000.0
                "itemStatus": true
            }
        ]
    }
]
